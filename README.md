## Overview

**Soccer Jeopardy** is a command-line trivia game written in **C** that tests the player's soccer knowledge across multiple categories and difficulty levels. Inspired by the Jeopardy format, the game allows users to choose a category, answer a question worth a certain number of points, and build or lose score based on correctness.

This project demonstrates core C programming concepts such as **functions**, **conditionals**, **loops**, **string handling**, and **user input processing** through a fun and interactive sports-themed application.

---

## Features

- Interactive soccer trivia game in the terminal
- Multiple trivia categories:
  - World Cup
  - Champions League
  - FC Barcelona
- Multiple point levels:
  - 100
  - 200
  - 300
- Score increases for correct answers
- Score decreases for incorrect answers
- Accepts multiple valid answers for some questions
- Case-insensitive answer checking
- Continuous gameplay while score remains non-negative

---

## Demo Flow

The game follows this workflow:

1. Launch the program
2. Choose a trivia category
3. Select a point level
4. Answer the question
5. Earn or lose points based on your answer
6. Continue playing until the score drops below zero
